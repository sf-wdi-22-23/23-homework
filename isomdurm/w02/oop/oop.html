What is the difference between a constructor and a prototype?

  • A constructor is the equivalent of a function in JavaScript. It sets the object's properties or method.
  • A prototype is a 'parent' object that is set up to allow you to assign certain methods to other objects that are related.

What does this mean inside a constructor?

Using this inside a constructor makes sure that the method is run on the current object.
