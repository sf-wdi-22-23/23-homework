What is the difference between a constructor and a prototype?

A constructor is a function created once the class is intiated. Where as a prototype acts as a class without needing to be defined. It is shared with multiple instances. Using the prototype method gives you a clean way to separate properties common to all instances and instance-specific ones.

Why do you want to use this in the constructor method?

It's mainly about structure and not wasting memory. You can add functions to an object in a constructor function.
