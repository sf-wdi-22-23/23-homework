Homework Questions
  • List three input types and give real world examples of when you might use each of them.
    • Drop-down lists - can be used in forms to select from preconceived options. Very frequently used to select security questions for password recovery.
    • file picker - used to select a file to upload somewhere or manipulate in some way.
    • radio - can be used on surveys perhaps to rate from 1-5.
  • In your own words, write definitions for client, server, and API.
    • Client - The computer who is making data requests (fetching) from the server by interacting with the interface.
    • Server - Where all the information is stored and is accessed (provided) when requested as it is consistently listening for a request event.
    • API - Simplifies web data organization, can be accessed by the client as it is stored in a server, and makes it easier to automate tasks.
