Three inputs types and when you might use them:

  1. Plain text input - a generic way of providing input
  2. Password field - allows you to enter input that echos dots rather than the letters, numbers, and symbols being typed.
  3. Check box - can be checked or unchecked (ie truthy or falsey)

Definitions:

  • client - a computer, phone, or tablet that is used to access and exchange information with a server.
  • server - an powerful machine in an uninteresting place which is the home of the bulk of the information one would want to access over the internet.
  • API - the interface between a client and a server that is used to communicate specified information between the two .
