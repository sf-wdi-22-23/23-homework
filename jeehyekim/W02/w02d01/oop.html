What is the difference between a constructor and a prototype?

A constructor is a function used for initializing new objects and you use a new keyword to call the constructor.

A prototype is an OOP model that doesn't use classes. It is used to attache properties and methos when you want to implement inhertance. You add methods and perperties on a function's prototype property to make those methods and properties available to instances of that function.


What does this mean inside a constructor?

The this keyword is used to define the properties and methods in the object definition and means "belonging to this object."
