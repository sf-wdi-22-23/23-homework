3 HTML5 input types with real world examples of usage:

Text area:
It is a customizable plain text field. This could be used in a comments section.
Check box:
This is either checked or unchecked. It could be used as a subscribed/unsubscribed feature.
Drop-down list:
When clicked, a list drops down with multiple options. This could be used to choose Mr., Ms., or Mrs.

API reading definitions:

Client:
The client is the end user's computer, phone, tablet, etc. that the user is requesting information with.
Server:
The server is a computer that contains information that can be served to clients.
API (Application Programming Interface):
Essentially a server that contains information that can be utilized by a third party. For example Yelp uses a Google Maps API to display a map within the Yelp app.
