Forms and API

Input Types
  • Checkbox: a checkbox input can be used to help with knowing what groceries are left to buy.
  • Drop-Down List: can be used to filter options. Multiple drop-boxes can be used together to further narrow down an item search.
  • Submit Button: can be used when submitting account information or to finalize an order online.
Definitions
  • Client: the user side of an integrated system that allows the user to know what data is available to manipulate.
  • Server: a more powerful computer, typically without a monitor, keyboard, or mouse, that provides the API to users upon request.
  • API: a program that allows a website's data to be digestible to a computer.
