  3. File picker inputs
    • Uploading photos to a website
    • Attaching documents to emails

Client, server and API

  • Client: When two systems link up, the side which is receiving the information through an API is the client.
  • Server: A server is a fast and powerful computer that programmers can connect to remotely. When two systems link up, the side which provides the data acts as a server.
  • API: An API is a Application Programming Interface. It is a tool that makes a website data digestible for a computer.
