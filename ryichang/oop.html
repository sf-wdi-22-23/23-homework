What is the difference between a constructor and a prototype?

The constructor is a method of the class. In Javascript the function serves as the constructor of the object, every action declared in the class gets executed at the time of instantiation. When using the prototype, method definitions are separated from the class.

What does "this" mean inside a constructor?

When this is inside a constructor, this is set to the object where the function is from. If this is calling from a variable, this can become a global object.
