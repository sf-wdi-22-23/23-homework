Questions and Things learned from the HTML and CSS readings
  1. In CSS, External stylesheet is the best > Internal for the lazy > Inline is the worst.
  2. In CSS, more specific selectors beat less specific ones id > class > tag. In most specificity value to least specificity value, style attribute > ID > Class, psuedo-class, attribute > element.
  3. When using CSS identify differences between tags, classes, and IDs. !Important trumps everything.
