Q: What is the difference between a constructor and a prototype?

A: A constructor is a function that will create a new instance of an object type, thereby constructing it.


Q: What does this mean inside a constructor?

A: It refers to the object that is being constructed, used to assign new key:value pairs, give it new properties,

or just refer to it to get properties/values for use in the function. Also keeps the references cleaner.

Makes it easier to read and also easier to change or use where you don't have to change every name in the code, just the first one.
