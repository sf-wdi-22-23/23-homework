    Q: In your own words, write definitions (no more than one sentence) for client, server, and API.
  • Client: The computer (usually) making the request to the server. Also the portal through which humans interface with the internet and servers.
  • Server: The larger, more powerful computer that is only programmed to recieve requests, and then send responses based on the instructions in those requests, and sometimes send a message back telling the client it can either not fulfill or not understand the request.
  • API: This is an Application Programming Interface, which I think describes its function well. They are datasets organized in a very specific way so that computer programs know how to parse them and extract the information they need, and they are usable for multiple clients in sometimes different ways.
